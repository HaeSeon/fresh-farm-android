# Fresh-Farm <농림축산식품부 데이터 활용 공모전 참가작>
가축 전염병 방지 어플리케이션.<br>
가축 질병 공공 데이터를 기반으로 최근 많이 발생하고있는 질병 데이터를 분석하여 동향 파악을 빠르게 하며 수 많은 데이터중 사용자에게 필요한 데이터만 추출해 맞춤 정보를 제공한다.
